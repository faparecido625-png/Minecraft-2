# MiniCraft Game

## Project Documentation
MiniCraft is a simple yet engaging craft game where players can explore, gather resources, build structures, and engage in various adventures in a blocky, pixelated world.

## Features
- **Exploration**: Explore vast landscapes with different biomes.
- **Crafting**: Gather resources to craft tools, weapons, and building materials.
- **Building**: Create structures and customize your environment.
- **Combat**: Fight against various creatures and defend your creations.

## Controls
- **W, A, S, D**: Move around
- **Left Mouse Button**: Use/craft tools
- **Right Mouse Button**: Place items
- **E**: Open inventory
- **Esc**: Open menu

## Setup Instructions
1. Clone the repository using `git clone https://github.com/faparecido625-png/Minecraft-2.git`
2. Navigate to the project folder: `cd Minecraft-2`
3. Install dependencies if applicable: `npm install` (or other instructions based on the setup requirements)
4. Start the game: `npm start` (or other instructions)
5. Enjoy playing MiniCraft!
